Linux Terminal Essentials

The 50 most important commands for system administration and daily use.

Navigation
pwd
Print Working Directory. Shows the path of the current directory.
pwd
Navigation
cd
Change Directory. Moves to a different folder.
cd /var/www/html
Navigation
ls
List contents of a directory (long format, show hidden).
ls -la
File Ops
touch
Create an empty file or update timestamp.
touch newfile.txt
File Ops
mkdir
Make Directory. Creates a new folder (including parents).
mkdir -p projects/python
File Ops
cp
Copy files or directories recursively.
cp -r source_folder/ backup/
File Ops
mv
Move or Rename files.
mv oldname.txt newname.txt
File Ops
rm
Remove files or directories forcefully/recursively.
rm -rf folder_name/
File Ops
file
Determine file type.
file archive.tar.gz
Viewing
cat
Concatenate and display file content.
cat config.json
Viewing
less
View file content page by page (for large files).
less large_log_file.log
Viewing
head
Output the first part of a file (e.g., 5 lines).
head -n 5 file.txt
Viewing
tail
Output the last part of a file and follow changes.
tail -f /var/log/syslog
Search
grep
Search for a pattern in text/files (case-insensitive).
grep -i "error" server.log
Search
find
Search for files in a directory hierarchy.
find /home -name "*.py"
Search
locate
Find files by name (uses a prebuilt database).
locate nginx.conf
Search
diff
Compare files line by line.
diff file1.txt file2.txt
Text Proc
sed
Stream Editor: substitute 'old' with 'new' globally.
sed 's/old/new/g' file.txt
Text Proc
awk
Pattern scanning and processing: print first column.
awk '{print $1}' data.csv
Permissions
chmod
Change file modes/permissions (rwxr-xr-x).
chmod 755 script.sh
Permissions
chown
Change file owner and group.
chown user:group file.txt
User
sudo
Execute a command as root (e.g., update package list).
sudo apt update
User
su
Substitute User. Switch to root account.
su - root
User
whoami
Print current user name.
whoami
System
uname
Print all system information.
uname -a
System
hostname
Show the system's IP address.
hostname -I
System
df
Disk Free. Report file system disk space (human readable).
df -h
System
du
Disk Usage. Estimate file space usage (summary, human readable).
du -sh folder_name
System
free
Display amount of free and used memory (RAM, human readable).
free -h
System
top
Display Linux tasks/processes interactively.
top
System
htop
Interactive process viewer (often preferred over top).
htop
System
ps
Report a snapshot of processes and filter for a specific one.
ps aux | grep nginx
System
kill
Terminate a process by PID.
kill 1234
System
shutdown
Halt/power-off the machine immediately.
shutdown -h now
Network
ping
Send ICMP ECHO_REQUEST to network hosts.
ping google.com
Network
ip
Show addresses on all interfaces.
ip addr show
Network
curl
Transfer data, show HTTP headers only.
curl -I https://example.com
Network
wget
Non-interactive network downloader.
wget https://example.com/file.zip
Network
ssh
OpenSSH remote login client.
ssh user@192.168.1.5
Network
scp
Secure copy (remote file copy).
scp file.txt user@remote:/tmp
Network
netstat
Print network connections (TCP/UDP, listening, numeric).
netstat -tulpn
Network
dig
DNS lookup utility.
dig google.com
Archives
tar
Extract files from a gzipped archive.
tar -xvf archive.tar.gz
Archives
zip
Package and compress files recursively.
zip -r mydata.zip folder/
Archives
unzip
Extract files from a ZIP archive.
unzip mydata.zip
Packages
apt
Package manager (Debian/Ubuntu): install git.
sudo apt install git
Packages
yum/dnf
Package manager (RHEL/CentOS/Fedora): update system.
sudo dnf update
Packages
pacman
Package manager (Arch Linux): sync and update.
sudo pacman -Syu
Editors
nano
Simple text editor.
nano config.txt
Editors
vim
Vi IMproved, a powerful text editor.
vim script.py
History
history
Show the command history and filter it.
history | grep "ssh"
Help
man
Access the manual page for a command.
man ls
Misc
echo
Display text and redirect to a file.
echo "Hello" > file.txt
Misc
alias
Create a shortcut for a command.
alias l='ls -la'
Misc
whereis
Locate the binary, source, and manual files for a command.
whereis bash
Misc
w
Show who is logged on and what they are doing.
w
Misc
date
Print or set the system date and time.
date "+%Y-%m-%d"
