SystemVerilog Quick Reference

1. Datentypen & Logik

SystemVerilog führt den universellen Typ logic ein (ersetzt reg und wire).

TypBeschreibung
bit2-state (0, 1), standardmäßig 0
logic4-state (0, 1, X, Z), standardmäßig X
int32-bit signed integer
byte8-bit signed integer
logic [7:0] data;  // 8-bit Vektor
bit clk;           // 1-bit 2-state
int i = 0;         // Ganzzahl

3. Prozedurale Blöcke

Spezialisierte Blöcke für Synthese-Klarheit:

Kombinatorik

always_comb begin
    if (sel)
        y = a;
    else
        y = b;
end

Sequentiell (Register)

always_ff @(posedge clk or negedge rst_n) begin
    if (!rst_n)
        q <= 8'h00;
    else
        q <= d_in;
end

4. Interfaces

Bündelt Signale für saubere Verbindungen.

interface bus_if(input logic clk);
    logic [7:0] addr;
    logic [31:0] data;
    logic enable;

    modport master (output addr, data, enable);
    modport slave  (input addr, data, enable);
endinterface

5. Control Flow & Arrays

Enums & Structs

typedef enum logic [1:0] {
    IDLE = 2'b00,
    BUSY = 2'b01,
    DONE = 2'b10
} state_t;

typedef struct {
    logic [15:0] addr;
    logic [7:0]  data;
} packet_t;

Arrays

int array [0:7];      // Statisch
int dynamic_arr [];   // Dynamisch
initial begin
    dynamic_arr = new[20];
    foreach (array[i]) array[i] = i * 2;
end

6. Verification (Testbench) Basics

initial begin
    clk = 0;
    forever #5 clk = ~clk; // Taktgenerator
end

initial begin
    rst_n = 0;

    assert (data_out == expected) 
        else $error("Mismatch!");
end
