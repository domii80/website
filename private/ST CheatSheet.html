Strukturierter Text (ST) CheatSheet

1. Grundstruktur & Syntax

  • Zuweisung: := (z.B. x := 10;)
  • Anweisungsende: Jede Zeile endet mit ;
  • Kommentare:
    • Einzeilig/Block: (* Kommentar *)
    • In modernen Systemen: // Kommentar
  • Groß-/Kleinschreibung: Case-Insensitive (meistens).

4. Variablen-Deklaration

VAR iCounter : INT := 0; (* Initialisierung *) rValue : REAL; bStart : BOOL; sName : STRING[20] := 'Maschine1'; END_VAR VAR_INPUT (* Eingangsvariablen *) END_VAR VAR_OUTPUT (* Ausgangsvariablen *) END_VAR VAR_IN_OUT (* Durchgangsvariablen *) END_VAR VAR_GLOBAL (* Globale Variablen *) END_VAR VAR_CONSTANT (* Konstanten *) END_VAR

6. Kontrollstrukturen

Bedingungen (IF / CASE)

IF bStart AND NOT bError THEN iState := 1; ELSIF bStop THEN iState := 0; ELSE iState := 99; END_IF;
CASE iState OF 1: // Tu dies 2, 3: // Tu das 4..10: // Bereich ELSE // Standard END_CASE;

Schleifen (FOR / WHILE / REPEAT)

// Festgelegte Anzahl FOR i := 1 TO 10 BY 1 DO aBuffer[i] := 0; END_FOR; WHILE i < 100 DO i := i + 1; END_WHILE;
// Mindestens einmal ausgeführt REPEAT i := i - 1; UNTIL i = 0 END_REPEAT; // Kontrolle: EXIT; // Schleife abbrechen

7. Komplexe Datentypen

Arrays (Felder)

VAR aNumbers : ARRAY[0..9] OF INT := [10(0)]; // 10 Integers, alle 0 aMatrix : ARRAY[1..3, 1..3] OF REAL; // 2D Array END_VAR // Zugriff: aNumbers[2] := 50;

Strukturen (DUT)

TYPE ST_Motor : STRUCT bOn : BOOL; rSpeed: REAL; END_STRUCT END_TYPE

8. Funktionsbausteine (FB) Aufruf

FBs müssen erst instanziiert werden (in VAR), bevor sie genutzt werden können.

VAR fbTimer : TON; // Einschaltverzögerung instanziieren END_VAR // Aufruf des FBs: fbTimer(IN := bStart, PT := T#5s); bOutput := fbTimer.Q; // Zugriff auf Ausgang
