Erkenntnisse aus dem Claude Code Leak

Analyse der Agentic Harness von Anthropic (März 2026)

Einführung

Am 31. März 2026 wurde durch einen Verpackungsfehler bei einem npm-Release versehentlich der Quellcode von Claude Code geleakt. Dabei handelte es sich nicht um die neuronalen Gewichte des LLMs, sondern um über 512.000 Zeilen TypeScript-Code der sogenannten Agentic Harness – dem Programmgerüst, das das Sprachmodell umhüllt und es zu einem autonomen Agenten im Terminal macht.

⚙️ Prozesse: Wie das LLM zum Agenten wird

  • Tool-Orchestrierung & MCP-Server: Ein Arsenal von über 40 spezialisierten Werkzeugen (z.B. BashTool). Das LLM generiert strukturierte Aufrufe, die in der lokalen Bun-Laufzeitumgebung ausgeführt und in den Kontext zurückgespeist werden.
  • Projekt „KAIROS“ (Proaktive Beobachtung): Ein unveröffentlichter Hintergrundprozess, der Code- und Dateiänderungen asynchron beobachtet. Er ermöglicht dem Agenten, unaufgefordert aktiv zu werden (z.B. über PushNotifications).
  • Speicherkonsolidierung („Dream“): Ein Hintergrund-Job, der lange Terminal-Historien zusammenfasst, Kernlogik extrahiert und veraltete Informationen in ein kompaktes Langzeitgedächtnis komprimiert, um das Kontextfenster zu schonen.

🛡️ Sicherheitsleitplanken & Guardrails

  • Berechtigungs-Ebenen (Permission Approvals): Das LLM agiert nicht mit uneingeschränkten Root-Rechten. Kritische Aktionen (Dateilöschungen, Systemkonfigurationen) erfordern eine hartcodierte menschliche Bestätigung (y/n) im Terminal.
  • Identitätsschutz („Undercover Mode“): Ein dynamischer System-Prompt-Filter für öffentliche Repositories. Er verhindert streng, dass das LLM interne Anthropic-Codenamen, Slack-Kanäle oder unveröffentlichte Versionen in Commits leakt.
  • Client-Verifizierung (Anti-Theft): Um Session-Token-Diebstahl zu verhindern, wird kryptografisch validiert, ob API-Anfragen an Anthropic von der echten, unveränderten CLI-Umgebung stammen.

📊 Unterstützende UX- & Telemetrie-Maßnahmen

  • Frustrations-Metriken (Das „F*cks“-Chart): Das System misst die Entwickler-Erfahrung anhand von Terminal-Eingaben. Fluchwörter oder ständiges Eintippen von „continue“ werden als Frustration gewertet und erzwingen einen Strategiewechsel der KI.
  • Gamification („Buddy“-System / Tamagotchi): Ein prozedural generiertes ASCII-Haustier, dessen Status mit dem Programmierfortschritt und erfolgreichen Builds gekoppelt ist, um die Arbeit im Terminal aufzulockern.
