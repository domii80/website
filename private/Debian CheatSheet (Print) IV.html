Debian Terminal Reference

Essential Commands & System Structure

Navigation & Files
Nav
pwd
Print working directory path.
pwd
Nav
cd
Change directory.
cd /etc/apt
Nav
ls
List files (long format, hidden).
ls -lah
Nav
tree
Show directory tree structure.
tree -L 2
Files
cp
Copy file or folder recursively.
cp -r src/ dest/
Files
mv
Move or rename files.
mv old.txt new.txt
Files
rm
Remove files (force/recursive).
rm -rf /tmp/junk
Files
mkdir
Create directory (incl. parents).
mkdir -p foo/bar
Files
touch
Create file or update timestamp.
touch lockfile
Files
ln
Create symbolic link (shortcut).
ln -s /orig /link
Files
file
Determine file type info.
file script.sh
Files
rsync
Smart file copy/sync tool.
rsync -av src/ dest/
View, Search & Process
View
cat
Output full file content.
cat /etc/issue
View
less
Scrollable file viewer.
less large.log
View
head
Show first 10 lines.
head -n 20 file.txt
View
tail
Show last lines / follow log.
tail -f access.log
Search
grep
Search text for pattern (regex).
grep -r "Error" .
Search
find
Search files by name/type.
find / -name "*.conf"
Text
sed
Stream editor (replace text).
sed 's/foo/bar/g' f
Text
awk
Process columns/fields.
awk '{print $1}' data
Text
sort
Sort lines of text.
sort names.txt
Text
uniq
Filter consecutive duplicates.
sort f | uniq -c
Text
wc
Count lines, words, chars.
wc -l file.txt
Text
diff
Compare files line by line.
diff file1 file2
Package Management (APT)
APT
apt update
Refresh package list index.
sudo apt update
APT
apt upgrade
Upgrade installed packages.
sudo apt upgrade
APT
apt install
Install a new package.
sudo apt install git
APT
apt remove
Remove package (keep config).
sudo apt remove nginx
APT
apt purge
Remove package AND config.
sudo apt purge nginx
APT
apt search
Search repo for packages.
apt search python3
APT
apt show
Show package details.
apt show curl
APT
apt autoremove
Remove unused dependencies.
sudo apt autoremove
DPKG
dpkg -i
Install local .deb file.
sudo dpkg -i app.deb
Systemd & Processes
Service
systemctl start
Start a service.
sudo systemctl start nginx
Service
systemctl stop
Stop a running service.
sudo systemctl stop nginx
Service
systemctl status
Check detailed status.
systemctl status sshd
Service
systemctl enable
Enable start at boot.
sudo systemctl enable ufw
Logs
journalctl
Query systemd journal logs.
journalctl -xe
Proc
ps
Snapshot of processes.
ps aux | grep java
Proc
htop
Interactive process viewer.
htop
Proc
kill
Kill process by PID.
kill -9 1234
Proc
pkill
Kill process by name.
pkill firefox
User & Permissions
Perms
chmod
Change permissions.
chmod +x script.sh
Perms
chown
Change file owner/group.
chown user:grp file
User
sudo
Run as superuser.
sudo -i
User
su
Substitute user identity.
su - username
User
passwd
Change user password.
passwd username
User
useradd
Create a new user.
useradd -m newuser
User
usermod
Modify user (groups).
usermod -aG sudo user
User
id
Print user/group IDs.
id www-data
User
w
Who is logged in.
w
Networking
Net
ip addr
Show IP addresses.
ip a
Net
ip route
Show routing table.
ip r
Net
ping
Test connectivity.
ping -c 4 8.8.8.8
Net
ss
Socket stats (ports).
ss -tulpn
Net
ufw
Firewall management.
ufw allow 22/tcp
Net
curl
Transfer data (HTTP).
curl -O http://s.com/f
Net
wget
Download files.
wget http://s.com/f
Net
ssh
Remote login (SSH).
ssh user@host
Net
scp
Secure Copy (SSH).
scp file user@host:~

Misc

Advanced editor.

Directory Structure
Dir / Root
/
Root directory. Start of hierarchy.
User: System
Dir / Bin
/bin
Essential user programs (ls, cp).
User: Everyone
Dir / Sbin
/sbin
System admin binaries (fdisk).
User: Admin
Dir / Config
/etc
Configuration files.
User: System/Admin
Dir / Home
/home
Personal user directories.
User: Standard Users
Dir / Root
/root
Superuser home directory.
User: Root
Dir / Var
/var
Variable data (Logs, Mail).
User: System
Dir / Tmp
/tmp
Temporary files (deleted on boot).
User: Everyone
Dir / Dev
/dev
Device files (disks, terminals).
User: System
