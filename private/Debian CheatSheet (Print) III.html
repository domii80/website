Linux Directory Structure
Dir / Root
/
Root directory. The start of the hierarchy. User: System
Dir / Bin
/bin & /usr/bin
Essential user programs (ls, cp). User: Everyone
Dir / Sbin
/sbin & /usr/sbin
System admin binaries (fdisk, reboot). User: Root/Admin
Dir / Config
/etc
Host-specific configuration files. User: System/Admin
Dir / Home
/home
Personal directories for users. User: Standard Users
Dir / Root Home
/root
Home directory for the superuser. User: Root (Admin)
Dir / Var
/var
Variable data (Logs, Web, Mail). User: System/Services
Dir / Temp
/tmp
Temporary files (deleted on reboot). User: Everyone
Dir / Dev
/dev
Device files (terminals, disks, usb). User: System
Dir / Mount
/mnt & /media
Mount points for external drives. User: Admin/Users
Dir / Proc
/proc
Virtual FS for system/kernel info. User: System (Read-only)
